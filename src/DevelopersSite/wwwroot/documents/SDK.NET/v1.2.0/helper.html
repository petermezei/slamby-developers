File Parser

File parsing and text extraction endpoint. Content must be provided as valid base64 string. Binary files can be easily loaded with File.ReadAllBytes.aspx) and converted to base64 string using Convert.ToBase64String.aspx) methods.

Example:

var manager = new HelperManager(configuration);
var fileParser = new FileParser
    {
        Content = "IkdvZCBTYXZlIHRoZSBRdWVlbiIgKGFsdGVybmF0aXZlbHkgIkdvZCBTYXZlIHRoZSBLaW5nIg=="
    };
var result = await manager.FileParserAsync(tag);
